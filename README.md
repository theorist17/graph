# graph

This project implements DFS, BFS, Djikstra algo.

## Geting Started

### Prerequites

Things you need to install this software

```
CMake
```
### Installing

A step by step series of commands to get a development env running

```
git clone https://github.com/theorist17/graph
cd graph
mkdir build
cd build
cmake ..
make
```

## Running the tests

### Run BFS & DFS

```
./traversal
```

### Run Dijkstra

```
./shortest_path
```
